For the Kronecker cases, files .tex, .pdf and .py are computed up to symmetry on the dimension (ex: mod $\mathfrak S_3$ for Kron-5-5-5).

The dominance inequalities are not given in these files.

The number given in the last column corresponds to the number of inequalities in these files.


The files .in and .out always contain all the inequalities (full orbits under symmetries for Kron INCLUDING the dominance inequalities in all cases).


Example:

The Kron-6-5-5 files deals with the inequalities defining the cone $$ Kron(\underline{d})= \left\{(\boldsymbol\lambda,\boldsymbol\mu, \boldsymbol \nu) : \begin{array}{l} \ell(\boldsymbol\lambda)\leq 6, \ell(\boldsymbol\mu)\leq 5,\ell(\boldsymbol\nu)\leq 5\\ \mbox{ and }\exists k>0\quad g_{k\boldsymbol\lambda, k\boldsymbol\mu, k\boldsymbol\nu}\neq 0 \end{array} \right\}\subset\{(\boldsymbol\lambda,\boldsymbol\mu, \boldsymbol \nu) : |\boldsymbol\lambda|=|\boldsymbol\mu|=|\boldsymbol\nu|\}. $$ One of the defining inequality is $$\lambda_2+3\lambda_3+4\lambda_4+5\lambda_5+6\lambda_6+6\mu_1+\mu_3+2\mu_4+3\mu_5+6\nu_1+5\nu_2+3\nu_3+\nu_5 \leq 12|\boldsymbol \lambda|$$ sometimes associated with its symmetric inequality $$\lambda_2+3\lambda_3+4\lambda_4+5\lambda_5+6\lambda_6+6\mu_1+5\mu_2+3\mu_3+\mu_5+6\nu_1+\nu_3+2\nu_4+3\nu_5 \leq 12|\boldsymbol\lambda|.$$ They appear in the different files (last lines) in the following form:


.pdf $(0, 1, 3, 4, 5, 6)\; (6, 0, 1, 2, 3)\; (6, 5, 3, 0, 1)\; - 12$
.py (0, 1, 3, 4, 5, 6, 6, 0, 1, 2, 3, 6, 5, 3, 0, 1, -12)
.in 0 -1 -3 -4 -5 -6 -6 -3 0 -1 -2 -5 -6 0 -1 -3 12
