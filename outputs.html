When the module moment_cone is installed, the files .py can be loaded in Sage.

The files .in and .out correspond to Normaliz inputs and outputs respectively. When available, the output file displays the extreme rays and a Hilbert basis of the corresponding cone


For the Kronecker cases, files .tex, .pdf and .py are computed up to symmetry on the dimension (ex: mod $\mathfrak S_3$ for Kron-5-5-5).

The dominance inequalities are not given in these files.

The number given in the last column of the table presented here is the number of inequalities in these files.


The files .in and .out contain all the inequalities (full orbits under symmetries for Kron) INCLUDING the dominance inequalities in all cases.


Example:
